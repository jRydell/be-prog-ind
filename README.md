# be-prog-ind

A CLI based program for event handling.
Using Commander, Inquirer and Axios.

## Set up
Clone and run npm install

## Commands

### Login

npm start login

username: johan
password: 12345

### List all events:

npm start list

### List event details:

npm start event-details

### Register user:

npm start register-user

### Remove user:

npm start remove-user
